Open-source AI system legitimacy checker. Test auditability, determinism, replayability, idempotency, and authority boundaries. Get PASS/FAIL with clear reasons. Built on deterministic kernel + admissibility. Think this test is wrong? Contribute and improve it.

# Can Your AI Survive an Audit?

Most AI systems work.

Very few can defend their decisions.

This project helps you test:

- Is your system auditable?
- Can decisions be replayed exactly?
- Are boundaries enforced?
- Is execution deterministic?
